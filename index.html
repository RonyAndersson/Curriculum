Desenvolvi um site de pontos turísticos da cidade, utilizando HTML para estruturação, CSS e Bootstrap para estilização responsiva e aprimorada, JavaScript para interatividade dinâmica, Java no back-end para processamento de dados, e integrei um banco de dados MySQL para armazenar informações sobre os locais. Aprender a conectar front-end e back-end para fornecer uma experiência coesa aos usuários foi um ponto-chave dessa experiência prática. Essa aplicação específica destacou a importância de uma abordagem abrangente para criar soluções web completas e envolventes.
